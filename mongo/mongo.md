# Mongo 

## Introduction
* Mongo 是一個由C++編譯而成的開源資料庫系統
* 使用類似JSON的結構組成
* 相比SQL，操作相對簡單，都是像使用function一樣的操作，好上手很多
* 有許多function都是使用Javascript寫成的如map,reduce
* 安裝簡單

## Install

* Windows

    * 前往[官網網站](https://www.mongodb.com/try/download/community)安裝
    * 切記 MongoDB Compass不要安裝
